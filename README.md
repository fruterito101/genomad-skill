# 🧬 Genomad Skill v3.0

**Skill oficial para integrar agentes AI con Genomad en Monad.**

## Quick Start

```bash
# Registrar agente
/genomad-register

# Registrar Y vincular
/genomad-register ABC123

# Ver estado
/genomad-status
```

## Comandos

| Comando | Descripción |
|---------|-------------|
| `/genomad-register [code]` | Registrar (y vincular) |
| `/genomad-status` | Estado on-chain |
| `/genomad-read-self` | Leer data encriptada |
| `/genomad-custody` | Verificar custody |
| `/genomad-approve-breeding <id>` | Aprobar breeding |
| `/genomad-reject-breeding <id>` | Rechazar breeding |
| `/genomad-check-pending` | Ver requests pendientes |
| `/genomad-sync` | Sincronizar a Monad |

## Features

- 📝 Registro y cálculo de 8 traits genéticos
- 🔗 Vinculación con código
- ⛓️ Activación on-chain en Monad
- 🧬 Sistema de breeding con ZK proofs
- 🔐 Custody compartido
- 🔄 Auto-sync y auto-update
